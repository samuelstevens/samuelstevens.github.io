Sharding

  • AstraBlog’s ‘Exploring Parallel Strategies with Jax’
  • Kaggle notebook ‘Parallelization and Distributed Training in JAX’

To perform sharding, we need four things:

  1. The logical mesh (the Mesh class)
  2. A partition spec describing how to partition an array across a device mesg (PartitionSpec)
  3. A memory layout (NamedSharding, which uses Mesh and PartitionSpec as inputs)
  4. A way to actually make arrays with the defined layout (jax.device_put(...))

It seems like there are a lot of repeated concepts: Mesh, PartitionSpec and Shardings.
