CUDA

I hate CUDA but it works better than any other hardware accelerator.

Here are live updated notes on how to deal with CUDA version mismatches and other various CRAP that stops me from doing interesting research.

Overview of CUDA

From this StackOverflow post.

CUDA has a runtime API and a driver API, each with their own version.

On Linux, the driver API is in libcuda.so and is installed by the GPU driver installer. The runtime API is in libcudart.so is installed by the CUDA toolkit installer (which might also include the driver installer).

nvidia-smi is part of the driver API and so is installed by the GPU driver installer. When you run nvidia-smi it will include the driver API version, which is not related to the CUDA runtime version.

nvcc is installed with the runtime API and only cares about the CUDA runtime API, not the driver API.
